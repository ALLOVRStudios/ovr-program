# OVR Program

OVR Program is a Solana Program which forms the basis of the ALLOVR Protocol.

Currently the only available functionality is:

- init
- mint
- trigger_inflation


## Build

    cargo build-bpf    

## Clean

    cargo clean

## Test

    cargo test-bpf
